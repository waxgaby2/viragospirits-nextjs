# 🍷 Virago Spirits

A modern luxury spirits ecommerce experience built with **Next.js, Tailwind CSS, and Sanity CMS**.

This project focuses on building a **premium brand experience**, not just a basic online store.

## ✨ About the Project

Virago Spirits is a conceptual luxury spirits brand website designed to simulate a real-world ecommerce experience.

The goal was to go beyond UI development and build a **content-driven, CMS-powered storefront** with a strong focus on branding, UX, and scalability.

---

## 🚀 Features

- 🛒 Fully functional cart system (add, update, remove items)
- 📦 Sticky order summary for improved UX
- ⚡ Responsive design (mobile, tablet, desktop)
- 🍸 Luxury-inspired UI with dark aesthetic
- 🧭 Mobile navigation with active route states
- 💳 Order summary breakdown (subtotal, tax, shipping)
- 🧠 Dynamic product content powered by Sanity CMS
- 🔄 Real-time content updates without redeploying

---

## 🧩 CMS Integration (Sanity)

This project uses **Sanity CMS** as a headless backend for managing product data and content.

### What is managed in Sanity:
- Product listings
- Product images
- Pricing
- Descriptions
- Categories

### Why Sanity:
- Allows non-technical content updates
- Scalable structure for ecommerce content
- Real-time content syncing with frontend
- Flexible schema for future expansion
